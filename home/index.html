Bootstrap Show Password

Bootstrap Show Password is a show/hide password plugin.

Download Getting started

Current version: v1.2.1

Designed for twitter bootstrap

Bootstrap Show Password has been designed to reduce development time and to require no specific knowledge from developers. It is both featherweight and feature-rich.

Features

  • Show/Hide password for a normal input element
  • Support all bootstrap version
  • Set placement of icon
  • Custom icons
  • Options can be passed via data attributes or JavaScript
